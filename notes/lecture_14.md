https://www.json.org/json-en.html  

CRUD -> Create, Read, Update, Delete

***RESTful API***

***STATELESS***

CLIENT
Protocol: HTTP/HTTPS
Request message:
1. Starting line
   1. http method: GET (Read), POST(Create), PUT and PATCH (Update), DELETED (Delete)
   2. url: baseUrl + endpoint
2. Header -> metadata
3. body

SERVER
Protocol: HTTP/HTTPS
Response message:
1. Starting line
   1. response code: 100, 200, 300, 400, 500
2. Header -> metadata
3. body -> JSON
